## fhtek-web-api-demo

fhtek web pages, each router is in a directory.

## How to run

### for dev

- install dependencies manually in each folder
- make dev CUSTOMER_ID=xxxx
  - CUSTOMER_ID should be oneof (demo fhtek)

### for prd

- make CUSTOMER_ID=xxxx MODEL_ID=xxxx
  - CUSTOMER_ID should be oneof (demo fhtek)
